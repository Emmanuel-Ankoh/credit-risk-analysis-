# credit-risk-analysis-
Credit risk prediction model using Python; Scikit-learn. Demonstrates anomaly detection, feature engineering, and risk scoring methodology, directly applicable to crypto exchange transaction monitoring and counterparty risk assessment for Bitfinex Risk Department.
##  Purpose
This project demonstrates **risk analysis and predictive modeling** skills for financial applications. Built for the Bitfinex Risk Department application.

##  Dataset
Home Credit Default Risk (Kaggle) - 32581 loan applications with default indicators

## Key Risk Metrics Analyzed
- Age vs default probability
- Income-to-loan ratios
- Employment stability
- Family size impact on repayment

## 🤖 Model Performance
- **ROC-AUC Score**: ~0.88 (Strong risk separation)
- **Features used**: 9 numerical +  4 categorical risk indicators
- **Algorithm**: Random Forest (interpretable, handles imbalance)

## Business Value for Exchanges
This same framework applies to:
- Anomalous trading pattern detection
- Counterparty credit risk scoring
- Fraud transaction identification

## Visualizations
- Risk factor distributions
- Feature importance (top risk drivers)
- ROC curve & confusion matrix

Python | Pandas | Scikit-learn | Matplotlib | Seaborn
